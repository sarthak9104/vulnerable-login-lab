# 🔐 Vulnerable Login Lab

A beginner cybersecurity project demonstrating real-world web vulnerabilities.

## 🚀 Features
- Vulnerable login system
- SQL Injection (Authentication Bypass)
- Session handling using cookies
- Tested using Burp Suite
